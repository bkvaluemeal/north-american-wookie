# north-american-wookie

## About
This project is an attempt at a bot to formulate the optimal balance of
ingredients for cheap food. The name was randomly generated by GitHub.

## Ingredients
* Masa harina
* Soy flour
* Oat Flour
* Dextrose
* Maltodextrin
* Potato starch
* Corn starch
* White rice flour
* Brown rice flour
